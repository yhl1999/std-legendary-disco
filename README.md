2.是否按要求输出字符 （比如是否少了个句号）

5.定义的最大值/最小值是否足够大/小
6.取余的MOD是否与题目所给一致
7.浮点常量精度是否够高，如pi的精度
8.字符串输入注意是否可能存在前置空格/前置0/空列
